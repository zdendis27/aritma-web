Abstrakt

Cílem projektu bylo vytvořit moderní webovou aplikaci pro SK Aritma Praha. Systém propojuje veřejný web, administraci, backend API a databázi pro správu klubového obsahu.

Přínos práce

Projekt ukazuje, že i menší sportovní klub může využívat moderní a dobře spravovatelné digitální řešení.

English Summary

This project focused on the development of a modern web application for SK Aritma Praha. The solution combines a public website, an admin interface, a backend API and a PostgreSQL database. The result is a responsive and modular system for managing club content.
